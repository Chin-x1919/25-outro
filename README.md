# 25-outro
python script that play XENOGENESIS-TheFatRat hook at perfect 00:00

---

## setup

- download music form [here](https://www.thefatrat.com/release/xenogenesis) (this code is not include music file)
- download this repo

- create venv(optional)
```bash
python3 -m venv venv
source venv/bin/activate
```

- app setup
```bash
pip install pygame
python3 setup.py
```

and boom done
